Squeezelite

Settings

For a full list of squeezelite command options visit the man page at https://ralph-irving.github.io/squeezelite.html

Friendly name that will appear in LMS
b:p:f:m
b Buffer time. If less than 500 then treated as buffer time in ms, otherwise size in bytes.
p Period count. If less than 50 treated as period count, otherwise size in bytes.
f Sample format 16|24|24_3|32
m Use mmap 0|1
Stream Buffer:DSP Buffer in kilobytes. Must be less than available RAM.
Default is 45, range is 1-99.
Default: flac,pcm,mp3,ogg,aac,alac,dsd
Available: flac,pcm,mp3,ogg,aac,wma,alac,dsd,mad,mpg
Default: -W -D 500 -R E -S /var/local/www/commandw/slpower.sh
-V [control] Use ALSA control for volume, otherwise use software volume.
-D [delay][:format] Output device supports DSD
[delay] = optional delay in ms switching between PCM and DSD
[:format] = dop (default if not specified), u8, u16le, u16be, u32le oe u32be.
-C [timeout] Close the output device after [timeout] seconds of the player being idle. The default is to always keep the device open as long as the player is "on"
-s [server][:port] Connect to the specified Logitech Media Server, otherwise use automatic discovery to find server on the local network. This option should only be needed if automatic discovery does not work, or the server is not on the local network segment e.g., behind a router.
