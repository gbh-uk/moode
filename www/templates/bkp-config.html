Options

Specify which parts of the system will be included in the backup zip file. A script file may optionally be added that performs custom actions during the restore process. It is run during moOde startup as part of restoring the Configuration settings. Note that WiFi and AP mode passwords will be included in the Config and Prefs part of the backup.

$_imported_scriptfile Add a script file that performs custom actions during the restore process.
$_togglebtn_backup_system Backup configuration settings and preferences including a backdrop image if any.
$_togglebtn_backup_camilladsp Backup CamillaDSP pipeline configurations and IR files.
$_togglebtn_backup_playlists Backup playlists.
$_togglebtn_backup_searches Backup saved searches.
$_togglebtn_backup_radiostations_moode Backup moOde supplied radio stations.
$_togglebtn_backup_radiostations_other Backup user added radio stations.
The backup zip file will be downloaded to the default Downloads folder.
Options

Specify which parts of the system will be restored. The on/off option buttons are automatically set based on the contents of the selected backup zip file. If a script file was added to the backup it will be run during moOde startup as part of restoring Config and Prefs.

$_togglebtn_restore_system Restore configuration settings and preferences including a backdrop image if any.
$_togglebtn_restore_camilladsp Restore CamillaDSP pipeline configurations and IR files.
$_togglebtn_restore_camilladsp_with_replace Remove existing configurations and IR files before restoring from backup.
$_togglebtn_restore_playlists Restore playlists.
$_togglebtn_restore_searches Restore saved searches.
$_togglebtn_restore_radiostations_moode Restore moOde radio stations.
$_togglebtn_restore_radiostations_other Restore user added radio stations.
$_imported_backupfile
Upload a backup zip file that will be used for the restore process. The RESET button will remove the uploaded file from the temp directory and set the option buttons to NO.
A restart is required if Config and Prefs are part of the restore.
